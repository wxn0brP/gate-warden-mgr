# Gate Warden Manager

A command-line tool for managing access control systems. Provides modules for user, role, ACL, RBAC, and ABAC management.

## Installation

[Ingr](https://github.com/wxn0brP/dotfiles)

```bash
ingr gate-warden-mgr
```

## Usage

```bash
gatewarden <module> [args]
```

Run `gatewarden` for detailed module information.
